- run Unit Tests, Integration Tests 
- do code quality checks
- do package applications and deploy

# Jenkinsfile:
- Scripted:
node {
	echo 'Build'
	echo 'Test'
	echo 'Integration Test'
}

- Declarative:
pipeline {
	agent any
	stages {
		stage('Build') {
			steps {
				echo 'Build'
			}
		}
		stage('Test') {
			steps {
				echo 'Test'
			}
		}
		stage('Integration Test') {
			steps {
				echo 'Integration Test'
			}
		}
	}
}	
		
- Post: always, success, failure, unstated, changed
